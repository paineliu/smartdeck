# Layout Grid

All slides use a 16:9 canvas.

Recommended page structure:

```text
+------------------------------------------------+
| Title                                          |
| Short statement                                |
|                                                |
| Main diagram / cards / metrics                 |
|                                                |
| Footer                                         |
+------------------------------------------------+
```

Principles:

- One page, one idea.
- One primary visual per page.
- Avoid paragraphs.
- Avoid overlapping arrows and text.
