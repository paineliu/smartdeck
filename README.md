# SmartDeck

SmartDeck is the business planning and investor presentation repository for **SmartBloom AI**.

## Positioning

**SmartBloom AI is a Language Intelligence Company.**

## Vision

Build the Language Intelligence Infrastructure for the AI Era.

## Mission

Empowering Every Language Expression.

## Build

```bash
cd investor
make
```

Output:

```text
investor/build/SmartBloom_AI_Investor_Deck.pdf
```
