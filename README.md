# SmartDeck

SmartDeck is a professional business presentation framework for SmartBloom AI.

This repository contains:

- `investor/`: SmartBloom AI investor deck
- `investor/theme/`: reusable Beamer theme and components
- `brand/`: design tokens and brand guidelines
- `docs/`: generated PDFs
- `.github/workflows/`: automatic PDF build

## Positioning

**SmartBloom AI is a Language Intelligence Company.**

## Vision

Build the Language Intelligence Infrastructure for the AI Era.

## Mission

Empowering Every Language Expression.

## Build

```bash
make
```

or:

```bash
cd investor
make
```

Generated PDF:

```text
docs/SmartBloom_AI_Investor_Deck.pdf
```
